My Harvard CS50x Final-Project

This Project was developed for Harvard's CS50x - Introduction to Computer Science, a MOOC course I completed in August 2022.

Idea and Technology

The task was to create any type of software solution that involved programming. Since I had been excited about learning Python and the Flask framework I decided to go for this relatively simple web application. I had also recently become a father and my wife kept writing down when our newborn would sleep, which inspired me to make this raw data a little more useful.

Functionality
Importing existing Data

First I needed to Import the Data I already had. This came in the form of a chat-log. I solved getting the data into the SQLite3 DB by reading the text-file with some Regular expressions. The code then saved the sleep and wake-up times as Date-Time objects in the database.

Users

As a basis I was able to use the authentication method from a previous CS50 prject. A table stores users, and their password hashmaps. Another table stores indivdual babies, which are connected to the correct user ID (in other words to each parent)

Tracking the Time

This was built using some JavaScript to get the local time and a simple HTML form which can be submitted to be handled by Flask and added to the Database

Statistical Data

To present the data I decided that a simple table would suffice. If there are enough dataentries for a selected baby, users can get the info on:

  • Average time spent napping per day
  • Average time spent sleeping at night
  • Average time per nap
  • Average bedtime
  • Average waketime
  • Top 3 Sleep times
  • Top 3 Sleep timeframes (+-10min)
